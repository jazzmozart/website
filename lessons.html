快速搭建nginx服务


  1. 获取镜像：docker pull nginx
  2. 生成容器：docker run -p:80:80 nginx
  3. 启动容器：docker container start <container-id>
  4. 停止容器：docker container stop <container-id>


其他命令

  1. 获取本地镜像列表：docker images
  2. 获取本地容器列表：docker container ps -a
